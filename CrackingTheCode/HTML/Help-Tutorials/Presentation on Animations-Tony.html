ANIMATIONS

    
What are CSS3 Animations?

    An animation lets an element gradually change from one style to another.
    But to have an animation you need keyframes.

The @keyframes Rule

When you specify CSS styles inside the @keyframes rule,
the animation will gradually change from the current style to the new style at certain times.

To get an animation to work, you must bind the animation to an element.

  The following example binds the "example" animation to the div element.
The animation will last for 4 seconds,
and it will gradually change the background-color of the div element from "red" to "blue".


  // The animation code //
@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}

// The element to apply the animation to //
div {
    width: 100px;
    height: 100px;
    background-color: red;
    animation-name: example;
    animation-duration: 4s;
}

  If the animation-duration property isn't specified, then the animation will have no affect
Because the default value is 0.
By using the keywords "from" and "to" they represent a progress bar as "from" is 0% and "to" is 100%.

You can also set a delay to the animation by adding animation-delay

So......
div {
  animation-delay: 2s;
}

Another property you can add is the animation-interation-count property
This specifies the number of times an animation should run.
So......
div {
  animation-iteration-count 4;
}

You can also reverse the direction and alternate cycles
To do this you need to add reverse or alternate in the animation-direction property.
So.....
div {
  animation-direction: reverse
  animation-direction: alternate
}


 animation-timing-function 

The animation-timing-function property specifies the speed curve of the animation.

The animation-timing-function property can have the following values:

ease - specifies an animation with a slow start, then fast, then end slowly (this is default)
linear - specifies an animation with the same speed from start to end
ease-in - specifies an animation with a slow start
ease-out - specifies an animation with a slow end
ease-in-out - specifies an animation with a slow start and end
